**Customer Churn Analysis**

Project Description:
Customer churn and revenue analysis project using SQL and Python to identify churn drivers, high-risk customer segments, and revenue loss patterns in a subscription-based telecom business. Generated data-driven insights and retention recommendations based on contract type, tenure, and pricing behavior.

Business Problem:
Subscription-based telecom businesses face recurring revenue loss due to customer churn.
This project analyzes historical customer data to understand who is leaving, why customers churn, and which customer segments create the highest revenue risk, enabling data-driven retention strategies.

Tools Used:
SQL (MySQL)
Python (Pandas, Matplotlib, Seaborn)
Jupyter Notebook
GitHub

Key Insights:
Around 27% of customers churn, creating significant recurring revenue risk.
Month-to-month contract customers show the highest churn (~43%), highlighting contract commitment as a major retention factor.
Customers within the first year show the highest churn, indicating early lifecycle engagement challenges.
Higher monthly charge customers show higher churn tendency, suggesting pricing or value perception issues.

Business Recommendations:
Provide incentives to migrate customers from monthly to long-term contracts.
Strengthen early customer onboarding and engagement programs.
Improve value perception of higher-priced plans through bundled services.
Use customer feedback analysis to identify key churn drivers.

Challenges Faced:
Handling missing and inconsistent values during data import and cleaning.
Converting grouped churn results into churn-rate metrics using SQL and pandas.
Translating raw analytical findings into clear business-focused insights and recommendations.

Key Learnings:
Strengthened practical skills in SQL querying, data aggregation, and pandas data transformation.
Improved ability to convert data analysis into business insights and retention strategies.
Gained experience building end-to-end data analysis projects, from data cleaning to executive recommendations.
